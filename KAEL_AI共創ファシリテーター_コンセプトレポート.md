# AI共創ファシリテーター ── 会議・研修・ワークショップの新しいサポート形態

> **対象読者：** 教育機関の研修担当 / 企業の人材開発・組織改革担当 / AX（AI Transformation）推進担当

---

## 🔑 エグゼクティブサマリー

従来の研修・会議では、**「場の対話」と「知識の記録・共有」は別々の作業**として扱われてきた。
対話は盛り上がっても、レポートが出来上がるのは数日後。その頃には熱量も記憶も薄れている。

**AI共創ファシリテーター**は、この問題を根本から解決する。
AIツールを活用し、「対話」と「知識化・共有」をリアルタイムで同時に行う**2人1組の新しいファシリテーション形態**だ。

---

## 📐 AI共創ファシリテーターとは

```mermaid
graph TD
    A[🤝 AI共創ファシリテーター<br/>2人1組のユニット]
    A --> B[🎤 ダイアログファシリテーター<br/>場を動かす・対話を引き出す]
    A --> C[🧠 AIナレッジファシリテーター<br/>場を記録する・知識に変換する・共有する]
    B --> D[参加者の対話・気づき・アイデア]
    C --> E[構造化レポート・図解・URL共有]
    D --> F[✨ 場の学びが即座に資産になる]
    E --> F
```

### 役割の定義

| 役割 | 担う機能 | 主なツール |
|------|---------|-----------|
| **ダイアログファシリテーター** | 場の空気を読む・問いを立てる・対話を深める・参加者の思考を引き出す | ホワイトボード・付箋・問いかけ |
| **AIナレッジファシリテーター** | 発言をリアルタイムで記録・AI処理・構造化・即時公開 | Zoom文字起こし・Claude Code・Obsidian・GitHub |

## 🔄 研修・ワークショップの進行フロー

```mermaid
sequenceDiagram
    participant D as 🎤 ダイアログFacilitator
    participant P as 👥 参加者
    participant N as 🧠 AIナレッジFacilitator
    participant G as 🌐 GitHub（全員共有）

    D->>P: 問いを立て、対話を促進
    P->>D: 気づき・アイデア・議論
    P-->>N: Zoom文字起こしログ（自動生成）
    N->>N: Claude Codeでレポート生成<br/>（図解・まとめ・キーワード抽出）
    N->>N: Obsidianに保存・整形
    N->>G: GitHubにプッシュ
    G->>P: URLを即時共有 ✅
    Note over P,G: 研修終了と同時に<br/>全員がレポートを持ち帰れる
```

## 📊 従来型 vs AI共創ファシリテーター型

```mermaid
graph LR
    subgraph 従来型
        A1[研修・会議] --> B1[手書きメモ・録音]
        B1 --> C1[担当者が後日文字起こし]
        C1 --> D1[数日後にレポート配布]
        D1 --> E1[😴 熱量・記憶が薄れている]
    end

    subgraph AI共創ファシリテーター型
        A2[研修・会議] --> B2[Zoom自動文字起こし]
        B2 --> C2[Claude Codeで即時構造化]
        C2 --> D2[当日中にURL共有]
        D2 --> E2[🔥 熱量のあるうちに振り返れる]
    end
```

| 比較軸 | 従来型 | AI共創型 |
|--------|-------|---------|
| レポート完成まで | 数日〜1週間 | **当日中** |
| 共有方法 | メール添付・印刷 | **GitHub URL 1本** |
| 図解・可視化 | 担当者のスキル依存 | **AIが自動生成** |
| 知識の蓄積 | ローカル保存・散逸 | **GitHubで永続公開** |
| 参加者の体験 | 「後で読んでください」 | **その場で一緒に確認** |
